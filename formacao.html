Toolkit Essencial

O que você precisa
dominar primeiro

Antes de qualquer especialização, existe um núcleo sólido que todo profissional de TI domina.

🧩

Lógica de Programação

A base de tudo. Antes de qualquer linguagem, aprenda a pensar algoritmicamente. Pseudocódigo e fluxogramas são seus primeiros aliados.

🐍

Python ou JavaScript

As linguagens mais demandadas do mercado. Python para dados e back-end; JS para web. Escolha uma e vá fundo — a segunda vem naturalmente.

🗄️

SQL — Bancos de Dados

Todo sistema guarda dados. SQL é a língua universal do armazenamento. É simples de aprender e abre portas em qualquer área da TI.

🌿

Git / GitHub

Controle de versão e portfólio público. Seu GitHub ativo é seu currículo real para o mercado. Todo commit conta.

# sequência sugerida para iniciantes:
▶ 01  Lógica     → Scratch / VisuAlg / pseudocódigo
▶ 02  Python    → CS50 / Curso em Vídeo / Alura
▶ 03  Git       → github.com + primeiro repositório
▶ 04  SQL       → SQLite → PostgreSQL
▶ 05  Especialização → Front / Back / Dados / Segurança

Sua Trajetória

Do zero à primeira vaga

Um caminho realista para quem está começando — sem atalhos, mas sem enrolação.

Fase 1 — Fundamentos

Aprenda a pensar em código

Lógica, algoritmos e uma primeira linguagem. 2 a 3 meses de estudo consistente.

Fase 2 — Especialização

Escolha sua trilha

Front-End, Back-End, Dados ou Segurança. Vá fundo em uma área antes de ampliar.

Fase 3 — Portfólio

Construa e mostre

Projetos no GitHub, um site pessoal e contribuições abertas. Prove que você sabe fazer.

Fase 4 — Mercado

Candidatura ativa

LinkedIn, ACATE Jobs, CIEE. Networking, entrevistas técnicas e seu primeiro contrato.

💼 Portfólio é seu currículo
